# Quest Log

Samsung-ready installable web app.

## Deploy with Netlify Drop
1. Run `npm install` and `npm run build` on a computer.
2. Upload the generated `dist` folder to Netlify Drop.
3. Open the resulting HTTPS link on your Samsung phone.
4. In Chrome: menu (⋮) → Add to Home screen / Install app.

Progress is stored locally on the device using localStorage.
